噗浪卡卡

2011 - 2016

當噗浪剛發表自訂表情符號功能時，即有不能超過60個的限制。忽然閃過一個可以突破限制的點子，就把它做成了Chrome外掛，最初只有6KB。原先只是自己使用同時分享給幾個好友測試，卻被意外瘋傳。後續不斷維護改進這個外掛，包含更流暢的快速操作介面、備份和使用經驗改善等等。

噗浪卡卡 至今仍是噗浪最受歡迎的必裝外掛，有超過四萬個噗友都喜愛這個！

Locate

2015

Locate 提供一個非常簡潔的介面讓你輸入目的地，APP會自動查詢出地點、電話以及導航代碼甚至行車時間。

有到過日本自駕旅遊都對 Denso Mapcode 導航代碼不陌生，它在行車導航上非常方便，但如果你不知道這代碼就會變得非常不方便。 這個影片示範了在日本開車如何設定導航。

網路上仍目前沒有一個方便查詢 Mapcode 的工具，上次出遊前儘管做足功課，但仍然碰到一時不知怎麼輸入某目的地的窘境。 沖繩回來後我花了一周時間從無到有研究 React.js 同時製作這個工具，在手機上直接輸入地點名稱就可以查詢出 Mapcode。當然事前規畫行程時也能在電腦上使用。

Sheepy 網頁複製機

2011-2015

完美儲存目前看到的頁面，甚至網頁應用程式或是登入中的臉書。
只要在頁面上按右鍵，點選「Clone this!」就可以把目前畫面儲存成單一HTML檔案。

因為瀏覽器並不能直接完整儲存動態網頁的狀態， Sheepy 是在 2011 年想到一套讓網頁狀態能攜帶的方法，包裝成Chrome擴充套件。

這個專案公開原始碼在 GitHub/chrome-sheepy

Eatgo Cafe

2015

Eatgo Cafe 可以使用台北捷運站或是店家特色來挑選咖啡廳。另外還針對手機版將的店家位置資訊簡化成「帶我去」按鈕，直接打開地圖APP導航。

當業務部門委託這個咖啡頻道需求時，我們幾乎沒有任何可用資訊。在資料與規範不齊之下，業務急需一個「產品」，而這產品的內容又是客戶。我最後決定將資料規範建立在 Google Spreadsheet ，以節省開發後台的時間。後續幾個月店家陸續簽約，業務只需依照習慣將資料填到試算表即可同步到網站上。

Eatgo 逸客民宿訂房系統

2014 - 2015

在逸客走吧旅行社工作期間，我負責開發前台訂房系統，從草創時期的3間民宿到30間、從陽春版到能符合各民宿的特殊規則，以及各式各樣的訂房附加服務，甚至同時多間房間預訂。我們的系統也許不算不上頂尖，但我們曾在線上訂房市場樹立了民宿里程碑。

KarmaPolice 卡馬警衛

2010 - 2011

卡馬警衛擁有自行開發的中文自然語言處理邏輯，也是噗浪上首個會與人和其他機器人連續對話的機器人。曾有超過五萬名粉絲，還與另一個有名的機器人－掰噗同時成為同人本主角，通稱【卡掰】。

最早是在玩 Plurk API 時，製作來用發噗當作指令，幫自己處理一些事情的機器人，例如在伺服器上執行指令，代號為 AI03。因為他會認主人，所以會對旁人回應一些話，後來被越來越多的人加為好友，於是發展成大眾互動的機器人。

目前已經停止運作和維護，使用手冊上可以了解更多細節。

Cloud Bookmark

2013

曾有一個人習慣把常用的網站寫在HTML，並當成首頁，他希望有一個簡單的入口頁能讓他管理連結。 為了讓操作更方便，必須能自動擷取網站資訊。當擷取到非Unicode編碼的網站資訊時，必須猜測它的原始編碼。想不到最好的解法，因此用了統計涵蓋某語言的字元範圍比例來猜測繁、簡、日文三種語言編碼。

Codeshare

2011

Codeshare 是大學的另一個畢業製作專題，提供一個可以線上測試與練習網頁開發的平台，提供 PHP 以及 HTML/CSS 兩種課程，以及可以安全執行 PHP 的沙盒環境。我們試著拋磚引玉製作只需完成「部分」且立即看到成果來達到學習目的，而且每個人都可以製作屬於自己的課程。

EmotionViz 聲色場所

2010

大學二年級在設計學院完成的團隊畢業製作。不同於主流的社群媒體，聲色場所使一個以聲音為基礎的社群，並且用色彩來記錄情緒變化。

決定製作這個時，完全沒想到錄製聲音會是最大難題，但答案就在當時最新版 Flash 10.1 beta 的 Alchemy 技術，可以使用 C-based LAME 編碼器。
